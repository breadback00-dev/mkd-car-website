# Project Context – Used Car Dealer Website

## Project Overview
This is a modern, clean, high-conversion website for a single-location used car dealer in Bristol.

Primary goal:
Generate phone calls and vehicle inquiries.

Keep it simple, fast, and trust-focused.

---

## Framework
Framework: Next.js (App Router)

Rendering Strategy:
- Default to Static Site Generation (SSG)
- Use server features only if absolutely necessary
- No API routes unless explicitly required

---

## Location (Critical)
Central Park, Unit 405D  
Petherton Rd  
Bristol  
BS14 9BZ  

Must clearly display address on:
- Homepage
- Contact page
- Footer

Include:
- Google Maps embed
- Get Directions button

---

## Design System

Theme: Light only

Background: #FFFFFF  
Alt background: #F7F7F7  

Primary color: Deep blue or charcoal

No gradients.
No aggressive sales UI.

---

## Conversion Priorities
1. Phone calls
2. Inquiry submissions
3. Clear location visibility

Clarity > Flash  
Speed > Animation  
Conversion > Novelty  

---

## Definition of Done
- Fully responsive
- Lighthouse > 85
- CTA visible above fold
- Address clearly visible
- Static rendering wherever possible
